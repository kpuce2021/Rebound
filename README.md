KPUCE Capstone Project 2021
====================
수면 관리 어플리케이션 (Sleep Management Application)
--------------------------------------------------------------------
### 설계 목적
* 사용자의 수면 데이터를 바탕으로 최적 수면시간, 수면환경을 확인 및 수면 가이드를 제공하는 어플리케이션 개발
### 팀 구성
* 조남준
* 이풍훈
* 김동윤
